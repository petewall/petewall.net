# petewall.net

This is the source for **[petewall.net](https://petewall.net)** — Pete Wall's personal blog and portfolio. The site is built with [Hugo](https://gohugo.io) using the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme and served as a static `nginx:alpine` container image.
